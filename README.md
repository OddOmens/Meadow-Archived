# Meadow

Meadow is a personal mental health companion app for iOS. It helps you track conversations with support people, set goals, capture thoughts, and manage your support network.

**Note: This project is archived and is no longer maintained.**

## Features

- **Conversations**: Log sessions with therapists, coaches, or friends.
- **Goals**: Set and track personal growth goals and treatment plans.
- **Thoughts**: Capture reflections and daily thoughts.
- **Supporters**: Manage your support network.
- **Privacy**: Local-first architecture with optional iCloud sync.

## Build Instructions

1.  Open `Meadow.xcodeproj` in Xcode.
2.  Ensure you have a valid development team selected (or specific team settings removed/configured).
3.  Build and run on your simulator or device.

## Requirements

*   Xcode 15.0+
*   iOS 17.0+
